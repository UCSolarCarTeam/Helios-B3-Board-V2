# Setup
- Clone the repository using ssh (recommended)
```
git clone git@github.com:UCSolarCarTeam/Helios-B3-Board-V2.git
```
- Update submodules
```
git submodule update --init
```
- Import into CubeIDE
  - Select File > Open Projects from File System
